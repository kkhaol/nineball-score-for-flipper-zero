# Nineball Score for Flipper Zero

一个专为 Flipper Zero 设计的中文九球追分计分器 App。  
支持 2 人 / 3 人追分模式、自动轮转、历史记录、胜局统计、震动反馈与台球桌风格 UI。

---

# 功能特性

## 🎱 九球追分规则

支持：

- 犯规（1分）
- 普胜（4分）
- 小金（7分）
- 大金（10分）
- 让球得分

程序会自动根据：

- 本家
- 上家
- 当前击球顺序

进行自动加减分。

---

# 👥 2人 / 3人模式

启动时可选择：

- 双人追分
- 三人追分

---

# 🔄 三人自动轮转

三人模式下：

- 胜利者 → 下一局第一个击球
- 失败者 → 第二个击球
- 剩余玩家 → 第三个击球

界面会自动重新排列玩家顺序。

---

# ✏️ 玩家昵称

支持：

- 自定义 3 字母昵称

例如：

TOM  
KEN  
JAY

---

# 📊 历史记录

自动保存：

- 每局结果
- 得分变化
- 胜局统计

---

# 🏆 胜局统计

自动记录：

- 玩家赢了多少局
- 当前总积分

---

# 📳 震动反馈

每次得分：

- Flipper Zero 会震动反馈

不同事件会有不同震动节奏。

---

# 🎨 台球桌风格 UI

界面采用：

- 深绿色球桌风格
- 大号分数字体
- 当前击球玩家高亮

更适合台球厅实战使用。

---

# 🎮 按键操作

| 按键 | 功能 |
|---|---|
| 上 | 犯规 |
| 右 | 普胜 |
| 下 | 小金 |
| 左 | 大金 |
| OK | 切换当前玩家 |
| 长按 OK | 撤销 |
| 长按 Back | 清零 |
| Back | 返回 |

---

# 📦 安装方法

将：

nineball_score.fap

复制到：

/apps/Games/

然后：

Applications → Games → Nineball Score

即可运行。

---

# ⚙️ 编译

适用于：

- Momentum Firmware
- 官方 Flipper Firmware

编译方式：

./fbt fap_nineball_score

或 GitHub Actions 自动编译。
